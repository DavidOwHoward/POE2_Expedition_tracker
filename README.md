# POE2 Expedition Tracker

A lightweight AutoHotkey utility for tracking Verisium Expedition encounters in Path of Exile 2.

## Current Features

- Layout tracking
- Remnant rune tracking
- Socketable rune tracking
- Currency tracking
- Compact GUI
- Encounter data model

## Planned Features

- Save encounters
- Discord webhook
- OCR for Waystones
- Session statistics
